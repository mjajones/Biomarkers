Assets Directory
================

This directory should contain the following image files for the app:

- icon.png (1024x1024px) - App icon
- splash.png (1284x2778px) - Splash screen
- adaptive-icon.png (1024x1024px) - Android adaptive icon
- favicon.png (48x48px) - Web favicon

You can create these images or use placeholder images for now.
For a quick start, you can use any square image renamed to these filenames.

To run the app without creating these assets, Expo will use default placeholders.
